# dota6.83

#### 介绍
Dota6.83版本移植，养老dota，best dota。

特性：
- AP模式
- 使用当前版本英雄池，去除天赋，去除魔晶，去除齐天大圣之后出的新英雄
- 英雄最高25级，使用6.83版本升级经验
- 使用6.83版本道具，不掉落中立道具
- 使用7.32地图，添加边路商店（树桩子那块），去除哨塔，去除野区赏金符，去除骑兵，去除塔回血加甲光环
- 使用6.83版本属性加成
- 使用6.83野区经济经验，30秒刷第一波野
- 信使需要购买，信使分享需要通过信使的技能开启，开启后无法关闭分享。信使在创建者到达4级后升级为飞行信使，移动速度和生命值不随升级改变。
- 背包道具拖上来后15秒才能起效，背包内道具不转CD
- 去除近战英雄内置伤害格挡，可以购买圆盾
- 重写塔防，塔防CD在一塔倒了之后刷新，塔防开启期间小兵没有无敌，没有溅射效果，塔防CD在泉水查看。
- 去除扫描
- 赏金符给个人经济经验，不再提供团队经济
- 随机英雄提供200金币
- 死后不提供免费TP

mod开源：https://gitee.com/bobo1993324/dota6.83
感谢SpellLibrary提供的部分自定义物品实现：https://github.com/Pizzalol/SpellLibrary

后续开发计划：
- 道具功能（补刀斧砍眼，散失驱散召唤单位，...）
- 去除英雄短位移技能 (拍拍踩地板，帕克沉默)
- 去除英雄清兵技能 (毒龙痰，小黑射箭)
- 去除英雄肉技能 (DP吸血，NEC虚无)
- 英雄属性成长改动
- 使用6.83地图（目前遇到中立单位无法喜迎小兵仇恨的问题）
- RD模式
- 法球叠加
